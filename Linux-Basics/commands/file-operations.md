# File Operations

## Objective

Learn basic Linux file manipulation commands.

---

## Commands Used

### Create Directory

```bash
mkdir testfolder
```

### Create File

```bash
touch file1.txt
```

### Copy File

```bash
cp file1.txt file2.txt
```

### Rename File

```bash
mv file2.txt newfile.txt
```

### Delete File

```bash
rm newfile.txt
```
