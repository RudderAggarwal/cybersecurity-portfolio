# Networking Commands

## Objective

Learn basic Linux networking commands.

---

## Commands Used

### Show Interfaces

```bash
ip a
```

### Display Listening Ports

```bash
ss -tuln
```

### Network Statistics

```bash
netstat -tuln
```

### Test Connectivity

```bash
ping -c 4 google.com
```

### Display Hostname

```bash
hostname
```

### Display IP Address

```bash
hostname -I
```
