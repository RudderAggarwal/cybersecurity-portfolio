# Permissions

## Objective

Learn Linux file permissions and how to modify them.

---

## Commands Used

### Create File

```bash
touch secret.txt
```

### View Permissions

```bash
ls -l
```

### Give Full Permissions

```bash
chmod 777 secret.txt
```

### Set Read and Write Permissions

```bash
chmod 644 secret.txt
```

### Restrict Permissions

```bash
chmod 600 secret.txt
```
