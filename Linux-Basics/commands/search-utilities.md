# Search Utilities

## Objective

Learn basic Linux search and text-processing commands.

---

## Commands Used

### Search for a String

```bash
grep root /etc/passwd
```

### Find Files

```bash
find /home/kali -name "*.txt"
```

### Locate Executables

```bash
which python3
```

### Count Lines, Words and Characters

```bash
wc /etc/passwd
```

### Display Beginning of a File

```bash
head /etc/passwd
```

### Display End of a File

```bash
tail /var/log/dpkg.log
```
