
# Navigation Commands

## Objective

Learn Linux filesystem navigation and directory listing commands.

---

## Commands Used

### Display Current Directory

```bash
pwd
```

### List Files

```bash
ls
```

### Detailed Listing

```bash
ls -la
```

### Change Directory

```bash
cd /home/kali
```

### Display Directory Tree

```bash
tree
```
