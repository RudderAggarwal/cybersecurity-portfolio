# Process Management

## Objective

Learn how to view and manage Linux processes.

---

## Commands Used

### View Processes

```bash
ps
```

### Detailed Process List

```bash
ps aux
```

### Interactive Process Viewer

```bash
top
```

### Start Background Process

```bash
sleep 300 &
```

### Search Process

```bash
ps aux | grep sleep
```

### Kill Process

```bash
kill $(pgrep sleep)
```
